三、核心页面原型与交互逻辑
1. 首页：我的职业镜像（Dashboard）
目的：一眼看清“我是谁”“差在哪”“下一步做什么”

模块设计（自上而下）：
① 职业身份卡

头像 + 姓名（可编辑）
动态标签：24岁 · 计算机专业 · Python熟练 · 求职中
职业倾向雷达图（5维度）：
技术型 85%｜管理型 30%｜创意型 60%｜服务型 40%｜创业型 50%
② 能力云图（动态词云）

硬技能：Python（熟练） SQL（掌握） Tableau（入门）
软技能：逻辑思维（强） 沟通表达（中） 抗压能力（待验证）
来源标注：点击“Python”可查看来自“简历解析+实训任务#123”
③ 成长进度条（目标导向）

主目标：成为初级数据分析师
进度：✅ SQL掌握 ✅ 项目经验 ❌ Tableau入门
提示：还需完成1个可视化实战项目
④ 快捷行动区（3按钮）

🔍 探索职业路径 → 跳转路径沙盒
🧪 开始AI测评 → 冷启动入口（新用户）
📌 今日推荐 → 个性化任务/课程（如“完成‘电商用户分析’微任务”）
✅ 差异化：所有数据实时更新，非静态简历。

2. 路径沙盒：未来推演（Path Sandbox）
目的：安全试错人生，降低重大决策焦虑

交互流程：
Step 1：设定目标

输入框：“我想在5年内成为______”
下拉推荐：产品经理 AI训练师 跨境电商运营（基于当前画像）
Step 2：生成3条路径（每条含4要素）

要素	示例
时间线	第1年：学Axure+实习 → 第2年：跳槽中级PM → 第3–5年：晋升
里程碑	✅ 主导1个DAU 10万+产品 ✅ 考PMP证书
成功率	82%（基于1,200名相似背景用户轨迹）
风险提示	⚠️ 当前缺乏用户调研经验 → 建议先完成“校园APP需求访谈”任务
Step 3：交互操作

拖拽调整：将“考研”节点拖入路径 → 系统重算时间线与成功率
对比模式：勾选路径A vs 路径B → 并排显示差异（收入曲线、压力指数等）
Step 4：一键启动

点击“开始这条路径” → 自动在成长中心生成学习计划+首期任务
🎯 技术支撑：职业知识图谱 + 因果推断模型 + 反事实验证（Counterfactual Validation）

3. AI合伙人：对话式中枢（Career Copilot）
目的：7×24小时职业顾问，替代所有菜单操作

典型场景：
用户输入	AI响应（含动作）
“我想转行做短视频运营”	1. 展示路径沙盒链接
2. 推荐课程《30天爆款脚本写作》
3. 附微任务：“为本地奶茶店写1条抖音脚本”
上传简历PDF	自动解析 → 更新职业镜像 → 提示：“建议强化‘数据分析’关键词”
“帮我找下周远程的数据分析兼职”	调用众包经纪人 → 返回3个高匹配任务（含预估报价）
“我刚做完飞书项目复盘”	调用生产力助手 → 提取成果 → 生成能力标签“跨团队协作（强）”
技术实现：
多模态理解：支持文本、语音、文件上传
Function Calling：可调用路径沙盒、成长中心、众包系统API
记忆上下文：记住用户目标（如“正在准备转行”）
💡 设计原则：90%的功能可通过对话完成，减少点击层级。

4. 成长中心：行动执行场（Growth Hub）
目的：将“推演”转化为“真实行动”，形成正循环

三大Tab整合逻辑：
Tab	内容	来源	关键创新
学习	个性化课程包
（如“30天成为数据分析师”）	超级学院平台	课程按路径沙盒目标动态生成
任务	企业真实微任务
（如“写618文案”“标注AI数据”）	B端合作企业	任务完成后自动更新技能标签
成就	证书、项目、徽章	自动归档 + 区块链存证	点击“项目”可生成作品集片段
核心机制：
自动归档：
接入钉钉/飞书日历 → 识别“项目结束会议” → 提示“归档本次成果？”
用户确认后 → 提炼能力标签（如“需求管理”） → 更新职业镜像
信用积累：
任务交付质量 → 影响众包经纪人接单优先级与报价能力
✅ 闭环验证：每完成1项，首页进度条更新，并推送新路径建议。

四、冷启动与数据融合策略
新用户激活路径：
极简入口：手机号注册 → 选择身份（学生/在职/自由职业）
快速画像（三选一）：
上传简历（PDF/Word）
10分钟AI测评（情景题+语音面试）
授权飞书/钉钉（仅读日历与文档标题）
生成V1镜像 → 引导设置首个目标 → 进入路径沙盒
多源数据融合表：
数据类型	来源	处理方式	更新频率
结构化经历	简历、社保（授权）	NLP实体抽取	一次性+手动更新
行为数据	平台点击、课程完成	埋点分析	实时
工作成果	飞书/钉头文档	LLM摘要提取	每周（需授权）
任务交付	众包平台	自动评分+标签	即时
外部信号	招聘网站（公开）	技能需求匹配	每日
隐私设计：所有敏感操作需二次确认，用户可随时删除数据。
